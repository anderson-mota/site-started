## Install

Para executar o projeto é necessário ter em sua máquina o [Node.js](https://nodejs.org/) e [Ruby](https://www.ruby-lang.org/pt/).

**Comando para instalar as dependências do projeto:**
 
```
npm install
```

## Usage

As tarefas do projeto são gerenciadas com o [GruntJS](http://gruntjs.com/), e básicamente é usado para executar o [Compass](http://compass-style.org/), executar o Uglify e executar o watch.

**Para iniciar o desenvolvimento usá-se o comando:**

```
grunt develop
```
